第5章 問題解決

  • 表計算ソフト(Excel)では = で数式を始める
  • 算術演算子
    • +, -, *, / のこと（足し算／引き算／掛け算／割り算）

参照（コピーするとき超重要）

  • 相対参照
    • コピーすると「同じ位置関係」でずれていく
    • 例：=A1 を右にコピー → =B1 になる
  • 絶対参照
    • コピーしてもセルの場所がずれない（$ を付ける）
    • 例：=$A$1 をどこにコピーしても A1 のまま
  • 複合参照（混合参照）
    • 行だけ／列だけ固定する
    • 例：=$A1（列A固定）、=A$1（行1固定）

関数

  • 関数
    • = の後ろに書く計算
    • 関数名の後ろの () の中に書いた 引数（ひきすう） を使って計算し、戻り値を表示する
    • 例：=SUM(A1:A10)
  • シミュレーション
    • 条件（入力）を変えたとき結果がどう変わるかを試す使い方
    • 綴り：simulation

並べ替え・抽出

  • 並べ替え
    • 昇順：値の小さい順
    • 降順：値の大きい順
  • 抽出
    • フィルター機能を使って必要なデータだけ取り出す作業

統計（表計算でよく使う統計量）

※ Excelの関数は基本 = を付けて使う（例：=AVERAGE(A1:A10)）

基本の統計量（代表的な関数）

  • 個数：=COUNT(範囲)（数値が入っているセルの個数）
  • 合計値：=SUM(範囲)
  • 最大値：=MAX(範囲)
  • 最小値：=MIN(範囲)
  • 平均値：=AVERAGE(範囲)
  • 中央値（メディアン）：=MEDIAN(範囲)
  • 最頻値（モード）：=MODE.SNGL(範囲)（複数ある場合は MODE.MULT）

四分位数・箱ひげ図で重要

  • 四分位数
    • 第1四分位数 Q1：データの下位25%の位置
    • 第2四分位数 Q2：中央値
    • 第3四分位数 Q3：データの下位75%の位置
    • Excel：=QUARTILE.INC(範囲, 1) / =QUARTILE.INC(範囲, 2) / =QUARTILE.INC(範囲, 3)
      • ※ QUARTILE.INC（包含）と QUARTILE.EXC（排他的）があるので、授業で扱った方に合わせる
  • 四分位範囲（IQR）：IQR = Q3 - Q1

ばらつき（分散・標準偏差）

  • 分散：平均からどれだけ散らばっているか（「平均との差」の二乗の平均）
    • Excel（母分散）：=VAR.P(範囲)
    • Excel（標本分散）：=VAR.S(範囲)
  • 標準偏差：分散の平方根（単位が元データと同じになる）
    • Excel（母標準偏差）：=STDEV.P(範囲)
    • Excel（標本標準偏差）：=STDEV.S(範囲)

度数分布表・ヒストグラム・箱ひげ図

  • 度数分布表
    • 値の範囲（階級）ごとに「いくつあるか（度数）」をまとめた表
  • ヒストグラム
    • 度数分布表を棒で表したグラフ（階級ごとの度数）
    • 棒グラフとの違い：ヒストグラムは（基本）連続した範囲を扱うので棒が隙間なく並ぶ
  • 箱ひげ図
    • 中央値、四分位数、最大／最小（や外れ値）で分布の様子を表す
    • 箱：Q1〜Q3、箱の中の線：中央値（Q2）

尺度水準（データの種類）

  • 質的データ（数の大小に意味がない／計算に向かないことが多い）
    • 名義尺度：分類だけ（例：血液型、都道府県、クラス）
    • 順序尺度：順番はあるが差の意味が一定でない（例：満足度1〜5、順位）
  • 量的データ（数の大小や差に意味がある）
    • 間隔尺度：差は意味があるが「0」が任意（例：気温℃）
    • 比例尺度：0に意味があり、比が意味を持つ（例：身長、重さ、時間）

時系列分析

  • 時系列データ：時間の順に並んだデータ（例：毎月の売上、日ごとの気温）
  • 平滑化：細かい上下動（ノイズ）をならして、傾向を見やすくする
    • 移動平均法：直近k個の平均をとる
    • 移動中央値法：直近k個の中央値をとる（外れ値の影響を受けにくい）

回帰分析

  • 回帰分析
    • データの関係を モデル関数（式）で表すこと
    • 一次関数で近似するのが 直線回帰、その直線が 回帰直線
    • 散布図で関係を確認してから使うことが多い
  • 最小二乗法
    • 予測値と実測値の差（残差）の二乗和が最小になるように回帰直線を決める方法
  • Excelでよく見る関数（出題範囲なら）
    • 傾き：=SLOPE(yの範囲, xの範囲)
    • 切片：=INTERCEPT(yの範囲, xの範囲)
    • 決定係数：=RSQ(yの範囲, xの範囲)

グラフの特徴（何を表すのに向く？）

  • 棒グラフ：カテゴリごとの量の比較
  • 折れ線グラフ：時間変化（時系列）の推移
  • 円グラフ：全体に対する割合（カテゴリ数が多いと読みにくい）
  • 帯グラフ：割合の比較（複数の集団で構成比を比べやすい）
  • レーダーチャート：複数観点の特徴（得意／不得意）を形で見る
  • 散布図：2つの量の関係（相関・回帰の確認）
